# **App Name**: AniHub Info

## Core Features:

- Core Info Display: Display detailed information about anime, manga, and novels.
- Recommendation Engine: AI tool that suggests similar anime, manga, or novels based on viewing history or selected title.
- Ranking Display: Display ranking, comment count, list additions, and star ratings for each title.
- List Management: Allow users to add titles to favorites and custom lists.
- Reutilizable components: Reusable UI components (cards) will apply the design of all data entities.

## Style Guidelines:

- Primary color: Saturated blue (#2962FF) for a modern and engaging feel.
- Background color: Light blue (#E5EFFF) to create a soft and clean backdrop.
- Accent color: Vibrant violet (#A214FF) for interactive elements.
- Headline font: 'Space Grotesk' sans-serif for titles; body font: 'Inter' sans-serif for descriptions. 
- Materialistic card design used consistently throughout the app for displaying information.
- Responsiveness: Ensure layout adapts to different screen sizes (mobile-first approach).
- Subtle transition animations on card hover and when loading content.
